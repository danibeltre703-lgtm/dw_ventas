# Data Warehouse - Ventas

Este proyecto consiste en la construcción de un Data Warehouse a partir de una base de datos de ventas.

## Fase 1 - Dimensiones
Se crearon y cargaron las siguientes dimensiones:
- dim_cliente
- dim_producto
- dim_fuente
- dim_tiempo

## Fase 2 - Tabla de hechos
Se creó la tabla fact_ventas y se realizó un proceso ETL que incluye:
- creación de tabla staging (stg_ventas)
- limpieza de datos
- carga de la tabla de hechos

## Tecnologías utilizadas
- MySQL
- SQL
